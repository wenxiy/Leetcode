* 如何从一个Activity跳转到另一个Activity

1. 新建一个Intent

```java
Intent intent = new Intent();
```

2. 建立跳转类与该类的联系

```java
intent.setClass(getApplication(),....Activity.class);
```

3. 跳转

```java
startActivity(intent);
```
